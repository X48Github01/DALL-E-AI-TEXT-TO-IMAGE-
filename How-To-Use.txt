- ขอเปิดใช้งาน API ของ OpenAI DALL-E ได้ที่ https://beta.openai.com/account/api-keys
- นำ API ที่ขอเปิดใช้งานมาใส่ใน config.ini
- prompt สามารถเพิ่มเติมลำดับ 2 3 4 ได้ โดยการใช้ \n คั่น เช่น Forest\nDark\nMood
